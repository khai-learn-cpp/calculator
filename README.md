
# Win32 Simple Calculator

## Build

### Prerequisite

 * MinGW >= 5.0.0

 * Node.js >= 5.0.0, with npm

 * Git >= 2.7.0

### Build steps

#### Clone this repo

```bash
mkdir calculator
cd calculator
git clone https://github.com/khai-learn-cpp/calculator.git .
```

#### Fetch dependencies

```bash
npm install
npm fetch
```

#### Run build script

```bash
npm build
```

## Run

Go to directory [out](./out) to find executable binaries
